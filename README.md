LINEと連携したタスク管理アプリです。  
公式アカウントにメッセージを送信することで、素早く、簡潔にタスクを登録することができます。  
公式アカウントから、現在のタスクを確認することも可能です。  
毎朝決まった時間に公式アカウントから残っているタスクが送られてくるので、その日のやることを整理して、1日を始めることができます。

## 機能一覧
・ ユーザー登録・ログイン機能 （LINEアカウントで管理）  
・ タスク管理機能（表示、登録、編集、削除）  
・ タスクのフィルター機能  
・ タスク完了ボタン  
・ LINEでのタスク登録機能  
・ LINEへのタスクの通知機能
